# Blackcoffer Dashboard v2.0

## Features
- Responsive design (mobile, tablet, desktop)
- Interactive Navbar with search, notifications, profile
- Collapsible Sidebar with filters
- Multiple pages (Dashboard, Analytics, Data Table, Reports, Settings)
- 10+ chart types
- Dark mode

## Quick Start
1. Copy jsondata.json to this folder
2. Make sure MongoDB is running
3. Run setup.bat (first time only)
4. Run start_backend.bat
5. Run start_frontend.bat
6. Open http://localhost:3000
